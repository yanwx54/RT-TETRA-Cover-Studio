## 1. 项目简介

RT-TETRA Cover Studio 是面向轨道交通 TETRA 专网的单基站覆盖距离计算与分析软件。

V1.0 目标是完成轻量化、可追溯、可导出的单基站覆盖计算工具，支持链路预算、路径损耗、最大覆盖距离、RSSI 曲线、覆盖曲线、参数敏感性分析和 Word/PDF 报告导出。

当前项目已形成 V1.0 文档体系初稿，已确定技术栈，并已实现核心计算闭环、四类传播模型初版、默认配置、四类标准算例、计算结果序列化接口、边界测试、GUI 原型初版、Path Loss/RSSI 曲线展示、基础交互完善和 Word/PDF 报告导出。报告已包含 Path Loss/RSSI 曲线图片、页眉页脚和工程信息，GUI 已接入 Word/PDF 导出按钮。
