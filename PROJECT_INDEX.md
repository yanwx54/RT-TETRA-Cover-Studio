## 1. 项目简介

RT-TETRA Cover Studio 是面向轨道交通 TETRA 专网的单基站覆盖距离计算与分析软件。

V1.0 目标是完成轻量化、可追溯、可导出的单基站覆盖计算工具，支持链路预算、路径损耗、最大覆盖距离、RSSI 曲线、覆盖曲线、参数敏感性分析和 Word/PDF 报告导出。

当前项目已形成需求文档和系统总体设计文档，并已创建基础工程目录。源码实现、测试用例、配置文件和报告模板尚未开始。

## 4. 下一步任务

优先级从高到低：

1. 补充 `README.md`
   - 内容：项目定位、目录结构、开发环境、运行方式、当前状态。
   - 验证：新协作者只看 README 能理解项目如何启动。

2. 确定 V1.0 技术栈
   - 内容：Python 版本、GUI 框架、绘图库、报告导出方案、测试框架。
   - 验证：`requirements.txt` 可据此填写。

3. 编写 V1.0 最小计算闭环
   - 内容：参数输入结构、EIRP、最大允许路径损耗、一个传播模型、覆盖距离求解。
   - 验证：至少一个示例输入能输出覆盖距离和中间计算过程。

4. 建立测试用例
   - 内容：链路预算、路径损耗、边界参数、非法输入。
   - 验证：`tests/` 下测试可一键运行并通过。

5. 补齐 GUI 与报告导出前的中间数据结构
   - 内容：统一结果对象，包含输入参数、公式、代入值、中间结果、最终结果。
   - 验证：同一份结果对象可同时支撑界面展示和报告生成。
