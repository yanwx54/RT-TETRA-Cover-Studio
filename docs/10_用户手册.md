# 10_用户手册

> 项目：RT-TETRA Cover Studio
>
> 版本：V1.0 Draft

---

## 1. 软件说明

RT-TETRA Cover Studio 是轨道交通 TETRA 单基站覆盖距离计算软件。

V1.0 用于完成：

- 链路预算；
- 路径损耗计算；
- 最大覆盖距离计算；
- RSSI 曲线；
- Path Loss 曲线；
- 参数敏感性分析；
- Word/PDF 报告导出。

当前项目尚未实现可运行程序。本文档先定义 V1.0 用户操作流程，后续随软件实现更新截图和具体按钮名称。

---

## 2. 适用用户

适用于：

- 轨道交通通信设计人员；
- TETRA 系统工程师；
- 招投标技术人员；
- 方案论证人员；
- 科研和教学人员。

## 3. 基本使用流程

```text
选择场景
↓
填写无线参数
↓
填写天馈参数
↓
填写场景参数
↓
设置覆盖设计参数
↓
点击计算
↓
查看结果和曲线
↓
导出报告
```

---

## 4. 参数填写

### 4.1 无线参数

| 参数 | 说明 |
| --- | --- |
| 工作频率 | TETRA 工作频点，单位 MHz |
| 基站发射功率 | 单位 W |
| 手台发射功率 | 单位 W |
| 基站接收灵敏度 | 单位 dBm，填写负值 |
| 手台接收灵敏度 | 单位 dBm，填写负值 |

### 4.2 天馈参数

| 参数 | 说明 |
| --- | --- |
| 基站天线增益 | 单位 dBi |
| 馈线损耗 | 单位 dB |
| 基站其他损耗 | 单位 dB |
| 手台天线增益 | 单位 dBi |
| 人体损耗 | 单位 dB |
| 基站分集增益 | 单位 dB |

界面中的单位位于参数名称左侧标签，输入框内只填写数字。

### 4.3 高度参数

| 参数 | 说明 |
| --- | --- |
| 基站高度 | 基站天线高度，单位 m |
| 手台高度 | 手持终端高度，单位 m |

### 4.4 场景参数

地面场景：

- 建筑平均高度；
- 建筑间距；
- 街道宽度。

高架场景：

- 高架高度；
- 曲线半径。

隧道场景：

- 隧道宽度；
- 隧道高度。

地下场景：

- 站厅宽度；
- 站厅高度；
- 墙体材料。

## 5. 计算结果说明

计算后显示：

| 结果 | 含义 |
| --- | --- |
| 最大覆盖距离 | 当前参数下可达到的最远覆盖距离 |
| 受限链路 | 上下行 MAPL 中较小的一方 |
| 系统 MAPL | 上下行 MAPL 的较小值 |
| 下行边界 RSSI | 覆盖边界的下行瞬时接收电平 |
| 使用模型 | 当前场景对应的传播模型 |

---

## 6. 计算过程查看

用户可查看：

- 基站和手台 EIRP 公式、功率换算和代入；
- 单侧正态分布阴影衰落余量和两端最低接收功率；
- 上下行 MAPL、系统 MAPL 和受限链路；
- 当前四类传播模型之一的实际公式和中间量；
- 覆盖距离数值求解结果。

这些内容可用于工程复核、技术交流和报告附件。

---

## 7. 曲线查看

V1.0 提供两类曲线：

- Path Loss 随距离变化曲线；
- RSSI 随距离变化曲线。

曲线上应标注覆盖边界，帮助用户判断覆盖极限位置。

---

## 8. 参数敏感性分析

计算完成后，系统分析关键参数变化对覆盖距离的影响。

默认分析：

- 发射功率增加；
- 天线增益增加；
- 基站高度增加；
- 工程裕度增加。

用户可据此判断优化方向。

---

## 9. 报告导出

支持导出：

- Word 报告；
- PDF 报告。

报告内容：

- 输入参数；
- 计算公式；
- 参数代入；
- 中间结果；
- 曲线图；
- 最终结论。

导出失败时，请检查：

- 输出目录是否存在；
- 文件是否被占用；
- 是否有写入权限；
- 文件名是否包含非法字符。

## 10. 常见问题

### 10.1 为什么不能计算？

可能原因：

- 必填参数为空；
- 数值超出范围；
- 场景参数未填写完整；
- 接收灵敏度填写为正数；
- 损耗或裕度填写为负数。

### 10.2 为什么覆盖距离很短？

可能原因：

- 发射功率低；
- 天线增益低；
- 馈线损耗高；
- 边缘覆盖率、阴影衰落标准差或干扰余量较大；
- 场景损耗高。

### 10.3 为什么报告导出失败？

可能原因：

- 文件被打开；
- 路径不可写；
- 文件名非法；
- 导出组件未安装。
