# 12_版本规划

> 项目：RT-TETRA Cover Studio
>
> 版本：V1.0 Draft
>
> 状态：初稿

---

## 1. 规划原则

版本规划遵循：

- 先完成可验证计算闭环；
- 再完成 GUI 和报告；
- 后续再扩展多站、GIS 和优化能力；
- 每个版本必须有明确验收标准；
- 不把后续版本能力提前塞入 V1.0。

---

## 2. V1.0 单基站覆盖计算

目标：

完成单基站、单场景、单次计算工具。

范围：

- 参数输入；
- 链路预算；
- 传播模型；
- 覆盖距离求解；
- RSSI 曲线；
- Path Loss 曲线；
- 参数敏感性分析；
- Word/PDF 报告导出。

验收标准：

- 四类场景可计算；
- 核心公式有测试；
- 单次计算不超过 3 秒；
- 报告可导出；
- 用户可按手册完成一次计算。

---

## 3. V1.1 工程参数完善

目标：

增强工程参数配置能力。

计划能力：

- 默认参数配置文件；
- 覆盖等级阈值配置；
- 工程裕度细分；
- 模型系数配置；
- 示例算例库。

验收标准：

- 修改配置无需改代码；
- 示例算例可回归测试；
- README 记录配置方式。

---

## 4. V1.2 报告与图表增强

目标：

提升报告可用性和图表表达能力。

计划能力：

- 报告模板；
- 图表样式优化；
- 导出 Excel；
- 报告结论自动生成；
- 多组参数对比。

验收标准：

- Word/PDF 报告格式稳定；
- 曲线标注覆盖边界；
- 报告可直接用于技术交流。

---

## 5. V2.0 多基站联合覆盖

目标：

支持多个基站共同覆盖分析。

计划能力：

- 多基站参数输入；
- 多站覆盖叠加；
- 最强信号判断；
- 基站间覆盖对比；
- 多站报告输出。

不在 V1.0 实现。

---

## 6. V3.0 GIS 地图显示

目标：

将覆盖结果与地图或线路图结合。

计划能力：

- 导入线路坐标；
- 地图底图显示；
- 覆盖区段展示；
- 站点位置管理；
- 地图结果导出。

前提：

- V2.0 多站覆盖稳定；
- 数据结构支持空间信息。

---

## 7. V4.0 覆盖优化

目标：

支持工程参数优化和站址辅助调整。

计划能力：

- 自动调整发射功率；
- 天线高度优化；
- 覆盖盲区识别；
- 参数组合比较；
- 输出优化建议。

---

## 8. V5.0 AI 辅助规划

目标：

结合历史项目和工程规则，提供智能规划辅助。

计划能力：

- 参数推荐；
- 方案解释；
- 自动生成技术说明；
- 历史案例检索；
- 风险提示。

前提：

- 有稳定算例库；
- 有可追溯计算报告；
- 有工程经验数据。
