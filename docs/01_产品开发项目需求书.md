# 一、项目简介

## 1.1 项目背景

轨道交通 TETRA 数字集群无线系统广泛应用于地铁、市域铁路、轻轨、有轨电车等通信系统。

在工程设计、招投标、方案论证及技术交流过程中，经常需要快速计算：

> 某一基站在当前参数配置下，最远能够覆盖多远？

目前业内主要依赖 Atoll、ICS Telecom 等大型无线规划软件进行计算，但软件价格昂贵、学习成本高，不适合日常方案设计及快速测算。

因此，开发一款轻量、专业、计算透明的 TETRA 单站覆盖计算软件，可满足日常工程设计需求，同时作为后续覆盖预测软件的核心计算引擎。

---

# 二、项目目标

开发一款针对轨道交通 TETRA 专网的单基站覆盖计算软件。

软件能够完成：

- 链路预算计算
- 路径损耗计算
- 最大覆盖距离计算
- 接收电平计算
- RSSI 曲线绘制
- 覆盖曲线绘制
- 覆盖等级评价
- 全过程计算展示
- Word/PDF 计算报告生成

整个计算时间不超过 **3 秒**。

# 六、核心计算模块

## ① EIRP

EIRP=发射功率+天线增益-馈线损耗-接头损耗

---

## ② 最大允许路径损耗

Maximum Path Loss=EIRP+接收天线增益-接收灵敏度-工程裕度

---

## ③ 路径损耗计算

自动调用：

- ITU Indoor
- COST231-WI
- Tunnel Model

输出：

Path Loss

---

## ④ 覆盖距离求解

采用数值迭代算法：

不断调整距离。

直到：

Path Loss=Maximum Path Loss

输出：

最大覆盖距离。
